## Objective
The objective of this project is to convert passive learning methods into active learning experiences using interactive simulations, visualizations, and computational models. The system aims to improve conceptual understanding by enabling users to actively engage with learning content rather than consuming it passively.

## Description
This project focuses on transforming traditional passive learning (reading, observing) into active learning by using Python-based simulations, physics models, visual tools, and interactive components. It includes multiple modules that demonstrate scientific, mathematical, and conceptual ideas through computation, visualization, and user interaction.

The project integrates physics simulations, 3D modeling scripts, UI components, and web-based interfaces to encourage experimentation, exploration, and hands-on learning.

## Project Structure
- app.py : Application entry or controller logic
- main.py : Main execution file
- physics.py : Physics-related simulations and logic
- newton.py : Newtonian mechanics concepts
- solar.py : Solar system simulation
- finalsolarsystem.py : Advanced solar system visualization
- generate_xyz.py : Coordinate generation utilities
- drums.py : Interactive or sound-based learning module
- blender_script_gen.py : Blender automation scripts for 3D learning
- export_glb_blender.py : Export 3D models for visualization
- webpage.py : Web-based learning interface
- ui1 : UI file for interactive learning
- templates/ : HTML templates for web UI
- requirements.txt : Python dependencies
